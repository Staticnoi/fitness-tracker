Workout Generator Logic:
Generate a workout plan based on:
- Goal
- Fitness level
- Focus areas
- Equipment
- Health issues
- Workout frequency
- Workout days

Rules:
- Beginner should get simpler workouts, lower volume, safer exercises.
- Intermediate should get moderate volume.
- Advanced should get higher volume.
- If user has knee issue, avoid jump-heavy and high-knee-stress exercises.
- If user has back or hernia issue, avoid heavy deadlift-style exercises.
- If user can't do jumps, avoid jumping exercises.
- If equipment is None / Bodyweight, only generate bodyweight exercises.
- If user has dumbbells, include dumbbell exercises.
- If user has barbells, include barbell exercises.
- If user has kettlebells, include kettlebell exercises.
- If user has machines, include machine exercises.
- If Full Gym is selected, use mixed gym exercises.
